# Programming Test Solutions

This repository contains solutions to 4 programming problems implemented in Python.

## Problems Solved

### Problem 1: Calculator Class
- **File**: `Program-1.py`
- **Description**: A calculator class that performs basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- **Input**: Two numbers (double) and operation type (string)
- **Features**: Error handling for division by zero and invalid operations

### Problem 2: Series Generator 1
- **File**: `Problem-2.py`
- **Description**: Generates a series of first 'a' odd numbers
- **Pattern**: 1, 3, 5, 7, ... for input 'a'

### Problem 3: Series Generator 2
- **File**: `Program-3.py`
- **Description**: Generates odd numbers series with special condition
- **Pattern**: For even input, generates series for (input-1); for odd input, generates series for input

### Problem 4: Multiple Counter
- **File**: `Program-4.py`
- **Description**: Counts how many numbers in a list are multiples of 1 through 9
- **Output**: Dictionary with counts for each divisor

## How to Run

1. Clone this repository
2. Run each program individually:
   ```bash
   python Program-1.py
   python Program-2.py
   python Program-3.py
   python Program-4.py
